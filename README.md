### TWAIN Bridge for Medi-6000 Plus Scanner ###
_ _ _

Currently, [Medi-6000 Plus Scanner](<http://www.microtekusa.com/products.php?KindID=6&ID=2>) does not support communication between its Data Source Driver and Windows 64-bit softwares. So I wrote this bridge to do this feature. Note: This is not a Data Source Driver.
